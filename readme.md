# TypeDo

## Functionality

### Syntax

> list | cmd | item

### Commands

Each Command should have 3 characters where the first one is always a dash -

| Command | Description          |
| ------- | -------------------- |
| -td     | Add a item item      |
| -xx     | Remove an item       |
| -clear  | Clear the whole list |

### Examples

```
general -td walk the dog
```

```
work -td eat lunch with stacy
```

## Lists

- General
- Work
- C1-4
